  public class Test004_HandlingProperties {

	public static void main(String[] args) throws IOException {
		// TODO Auto-generated method stub
 FileInputStream ob = new FileInputStream("F:\\java\\Java\\Java\\Java\\FileHandling\\TestProperties.properties"); // stream to connect your properties file to object of property.
 Properties obj = new Properties();
 obj.load(ob);                                               // to load or call the fileinputSTream to property file.
 System.out.println(obj.getProperty("name"));  
 System.out.println(obj.getProperty("class")); 
 System.out.println(obj.getProperty("id")); 
 System.out.println(obj.getProperty("location"));               // getProperty method to print you property value into your console.
	} 

 }
to read the data into the properties to use FileInputStream.
To write the data into the Property to use FileOutputStream.


ex, 
  public class Test004_WriteProperties {

	public static void main(String[] args) throws IOException {
		// TODO Auto-generated method stub
FileOutputStream d = new FileOutputStream("F:\\java\\Java\\Java\\Java\\FileHandling\\NewFile.properties");
Properties f = new Properties();
// to write the data into the properties file the keyword called f.setproperty()
f.setProperty("mark1"," 20");
f.setProperty("mark2"," 30");
f.setProperty("mark3"," 40");
f.store(d, "Student Marks");  // to store the data into the properties file.

	}

}


				 	read and write in excel file.

Here no inbuild class in jave so, we use third party tool.
.Apache they will provide a toll called Xls_Reader.

in this method to print how many row in the excel and coloum  the method called.

   - getRowCount()
   - getColumnCount().

public class Test005_handlingExcel {

	public static void main(String[] args) {
		// TODO Auto-generated method stub
Xls_Reader ob = new Xls_Reader("F:\\java\\FileHandling\\excel.xlsx");
System.out.println(ob.getRowCount("record"));
System.out.println(ob.getColumnCount("record"));
System.out.println(ob.getCellData("record", 0,3));  // to print the data in the excel to console.	
ob.setCellData("record","Rollnumber", 5 ,"25");   // to write into the excel sheet
}
